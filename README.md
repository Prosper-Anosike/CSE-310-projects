# Overview

As a software engineer, I am continuously building practical software projects to strengthen my programming skills and establish effective development workflows. This project served as the foundation for future software development by confirming that my programming environment, version control, and documentation process are all working correctly.

This software is a simple **Hello, World!** application written in Python. When executed, it prints a greeting to the console. Although the program is intentionally simple, it demonstrates the complete software development lifecycle—from writing and testing code to documenting the project and publishing it to GitHub.

The purpose of this project is to establish a reliable development workflow that I can use for increasingly complex applications throughout my software engineering journey. By completing this project, I verified my development environment, practiced using Git and GitHub, and reinforced the importance of clear documentation.
